### How to install ###

* Clone repo

```
git clone https://github.com/tab/vimfiles.git ~/.vim
```

* Clone [vundle](https://github.com/gmarik/vundle)

```
git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle
```

* Add to your .vimrc

```
echo 'source ~/.vim/vimrc' >> ~/.vimrc
```

* Run Vim and execute

```
:BundleInstall
```
